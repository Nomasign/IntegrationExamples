# Step 1 — FAQ

Common questions about creating a NomaSign account and enabling the Integration feature.

---

### What plans include the Integration feature?

Integration is available on Business and Enterprise plans. If you're on a Starter or Free plan, you'll need to upgrade. Check **Account Settings → Subscription** for your current plan details.

### Can I test without paying per API call?

Yes. API calls are included with the Integration feature — there's no per-call billing. You can test freely by sending documents to your own email addresses.

### Is there a sandbox or test environment?

No. NomaSign operates in production only. There's no separate sandbox URL or test mode. You test by sending real documents to real email addresses (use your own).

### Can multiple people in my org use the Integration feature?

The Integration feature is an accout (tied to your plan), but API access is granted per-user via the **Integrator** role. You can have multiple integrator accounts, each with their own credentials and templates.

### Do I need a credit card to start?

This depends on the plan and any trial period. Check [nomasign.com/pricing](https://nomasign.com/pricing) for current offerings.
